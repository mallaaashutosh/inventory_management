# inventory_management
A simple terminal based python program for small stores to generate invoice and keep track of their sales and restock. 

## Main Menu Of our Program
This is the main display of our project. We Input numbers according to our requirement. 
<img width="1918" height="411" alt="image" src="https://github.com/user-attachments/assets/7c314af2-e3ef-45cf-9b41-b58bc3998e90" />

## View Products
Input number 1. Through this we can see different products available in our store. This is for customers. 
<img width="1918" height="1037" alt="image" src="https://github.com/user-attachments/assets/220a4f1b-9429-43ac-a091-b0d74ef91442" />

## Purchase Products
Input number 2. Through this we can purchase products. We have to input the given products name. If the product is not available it will say "product not available". 
After a sucessful purchase, It will automatically generate a invoice. 
<img width="1918" height="1051" alt="image" src="https://github.com/user-attachments/assets/6b534074-e19a-47db-947c-8c9f8acb6f9b" />

## Restock Products
Input number 3. This is for the vendor. If the vendor wants to restock products, simply input the stores name and products details like name, quantity and price. After the restock, it will
generate an invoice. 
<img width="1918" height="1045" alt="image" src="https://github.com/user-attachments/assets/3e9b3b35-b15f-4a4c-9c51-8550e7ae31e2" />

## Save and Exit
Input number 4. Exit the program.
